# 🐹 golang-studies-2026

Saindo do .NET e mergulhando de cabeça no Golang.

# 🧭 Sobre o projeto
Esse repositório é meu caderno de estudos de Go.
Venho do ecossistema .NET (C#) e tô aproveitando essa jornada pra entender não só a sintaxe, mas a filosofia por trás do Go — concorrência nativa, simplicidade forçada, performance sem cerimônia.
A ideia é cobrir desde os tipos mais básicos até os padrões mais avançados, documentando o caminho.

## 🆚 .NET → Go: Mapa mental
 
| Conceito .NET | Equivalente em Go |
|---|---|
| `class` | `struct` + métodos |
| `interface` | `interface` (implícita!) |
| `IEnumerable` | `slice` / `channel` |
| `async/await` | `goroutine` + `channel` |
| `try/catch` | retorno de `error` |
| `null` | zero values + ponteiros |
| `generics (C#)` | generics (Go 1.18+) |
| `namespace` | `package` |
| `NuGet` | `go get` / `go.mod` |

## 💡 Filosofia do projeto
 
- Um conceito por pasta, simples e focado
- Comentários em código explicando o **porquê**, não só o **o quê**
- Comparações com C# onde fizer sentido
- Sem frameworks, sem magia — só Go puro
